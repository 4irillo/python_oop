# ЛР-4 — Абстрактные классы

## Цель работы

* Изучить абстрактные классы (ABC) в Python.
* Реализовать интерфейсы через `@abstractmethod`.
* Применить принцип подстановки Барбары Лисков.
* Работать с объектами через интерфейс, а не через конкретный класс.
* Использовать `isinstance()` для проверки реализации интерфейса.

# Реализованные классы

В рамках лабораторной реализованы два абстрактных интерфейса и две модели, их реализующие.

### Интерфейсы:

```
Printable
Comparable
```

### Модели:

```
Student(Printable, Comparable)
Course(Printable, Comparable)
```

### Реализовано:

* Два абстрактных класса с методами `@abstractmethod`
* Класс `Student` — реализует оба интерфейса
* Класс `Course` — реализует оба интерфейса
* Коллекция `ItemCollection` с фильтрацией по интерфейсам
* Функции, работающие через интерфейс (`print_all`, `find_max`)

