# Реализованный класс

В рамках лабораторной реализован класс **TeacherCollection** — коллекция для управления объектами `Teacher` из ЛР-1.

### Реализовано:

* Базовые операции (`add`, `remove`, `remove_at`, `get_all`, `get`)
* Поиск по критериям (`find_by_name`, `find_by_subject`, `find_by_rank`)
* Фильтрация (`get_active`, `get_inactive`, `get_by_subject`, `get_top_rated`)
* Сортировка (`sort_by_name`, `sort_by_age`, `sort_by_score`, `sort`)
* Магические методы:
  * `__len__` — размер коллекции
  * `__iter__` — итерация по коллекции
  * `__getitem__` — доступ по индексу и срезу
  * `__contains__` — проверка вхождения
  * `__repr__` — строковое представление
