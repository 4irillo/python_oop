# Реализованные классы

В рамках лабораторной реализована иерархия классов на основе модели из ЛР-1.

### Иерархия:

```
BaseTeacher
├── Professor
└── Lecturer
```

### Реализовано:

* Базовый класс `BaseTeacher` с общими атрибутами и методами
* Дочерний класс `Professor` с полями:
  * `research_area` — область исследований
  * `publication_count` — количество публикаций
* Дочерний класс `Lecturer` с полями:
  * `hours_per_week` — часы в неделю
  * `contract_type` — тип контракта
* Переопределенные методы (`get_info`, `__repr__`)
* Полиморфный метод `calculate_salary()`
* Коллекция `MixedCollection` с фильтрацией по типу

# Структура модулей

```python

└── BaseTeacher (класс)
    ├── __init__()
    ├── activate()
    ├── deactivate()
    ├── get_info()
    ├── __repr__()
    └── __eq__()

models.py
├── Professor(BaseTeacher)

│   └── __repr__() (override)
├── Lecturer(BaseTeacher)
```
